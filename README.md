A real-time web app that helps users discover and chat with nearby online friends — powered by Firebase (Auth, Firestore, and Realtime Database).
Users can update their live location, view who’s nearby within a defined distance, send chat requests, and instantly message once connected.

*** Features ***

✅ User Authentication — Secure Firebase Email/Password login & signup
✅ Profile Setup — Name, age, about, and automatic geolocation
✅ Live Online Status — Uses Firebase Realtime Database to track online/offline state
✅ Nearby Discovery — Shows users within 25–1000 km using Haversine formula
✅ Chat Requests — Send, receive, accept, or reject requests in real time
✅ Instant Chat — Real-time messaging powered by RTDB
✅ Recent Chats — Displays your latest conversations
✅ Auto Location Update — Background location refresh every few minutes
✅ Responsive UI — Works across desktop and mobile
✅ Version Tracking — Shows app version and checks for updates
